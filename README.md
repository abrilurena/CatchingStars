# 🌟 Catching Stars
Ever wanted to start your day with a little magic? Catching Stars is a small app made with love for little ones.
Children can pick a theme they love — like the sea, the forest, or the city — and receive a sweet, positive affirmation just for them. Each affirmation appears on a beautiful canvas with a matching background, making every message feel like a little story.

## How it works

- Choose a theme you love
- Wait for your affirmation ✨
- Read it, believe it, and go conquer the day! 🚀

## Built with

- Python
- AI (ChatGPT)
- Graphics Canvas


This project was built as part of the Code in Place 2026 final project — Stanford University's Python course. 🎓
